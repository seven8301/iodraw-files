sequenceDiagram
    participant User

    FrontEnd->>API Gateway: HTTP GET /travel/info (View Travel Information)
    activate API Gateway

    activate Lambda
    Lambda->>DynamoDB: Query travel data
    activate DynamoDB

    deactivate DynamoDB
    Lambda->>API Gateway: Send response
    deactivate Lambda

    deactivate API Gateway

    deactivate Lambda
