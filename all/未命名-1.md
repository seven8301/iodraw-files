```mermaid
@startuml
actor Employee
participant "Recreation and Wellness System" as System
participant HR
participant "Incentive Database" as IncentiveDB

Employee -> System: Register for recreational program
System -> System: Store registration data
System -> HR: Notify HR about new registration
HR -> System: Confirm employee participation

loop Every Week
    System -> Employee: Send participation reminder
    Employee -> System: Log participation
end

System -> IncentiveDB: Check eligibility for incentive
alt Eligible for Incentive
    System -> HR: Notify HR to provide incentive
    HR -> Employee: Offer incentive
else Not Eligible
    System -> Employee: No incentive awarded
end

@enduml

```
